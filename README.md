# CajaDeAhorros

  Programa desarrollado en Java a modo de caja de ahorros. Útil para registrar ingresos y gastos con un campo autocalculado con los 
totales de la caja. Incluye un gráfico para visualizar la evolución del capital en el tiempo.

  Java program which acts as a savings bank. Useful for keeping track of incomes and outcomes with a self-calculated field which contains
the total amount of the "bank". It includes a line chart to visualize the balance evolution in time.
